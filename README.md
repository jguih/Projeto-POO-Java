# Simulador de Corrida em Java
![2021-12-30 00_54_51-Window](https://user-images.githubusercontent.com/62079543/147720964-3e11bec6-754b-48de-9801-9b3d76a698a2.png) 

Um simples simulador de corrida compilado no terminal feito como projeto final da disciplina de Programação Orientada à Objetos.
Todos os inputs são validados, mesmo que o usuário informe um charactere inválido o programa continua funcionando e pede o input novamente.
